# Tello无人机多功能控制系统

## 项目简介

这是一个基于Python和PyQt5开发的Tello无人机多功能控制系统，集成了多种先进的计算机视觉和人工智能技术，实现了无人机的多种控制方式和智能跟踪功能。

## 功能特性

### 1. 多种控制方式
- **键盘控制**：使用键盘快捷键控制无人机飞行
- **语音控制**：通过中文语音指令控制无人机
- **手势控制**：通过摄像头识别手势控制无人机
- **体态控制**：通过人体姿态识别控制无人机

### 2. 智能跟踪功能
- **绿球跟踪**：自动识别并跟踪绿色球体
- **人脸跟踪**：自动识别并跟踪人脸

### 3. 其他功能
- 实时视频流显示
- 飞行状态监控（高度、速度、电量、WiFi信号）
- 拍照和录像功能
- 紧急制动功能


## 文件功能说明

### 1. main.py
**功能**：
- 主程序入口，基于PyQt5的图形用户界面
- 无人机连接管理和状态监控
- 实时视频流显示
- 键盘控制实现
- 飞行状态数据显示（高度、速度、电量、WiFi信号）
- 拍照和录像功能
- 紧急制动功能
- 集成语音控制、手势控制、体态控制等子模块

**主要类**：
- `mainWindow`：主窗口类，包含所有UI和功能逻辑

### 2. myTello.py
**功能**：
- 继承自tellopy库，扩展无人机功能
- 实现视频流的实时解码和处理
- 支持自定义命令发送
- 多线程视频处理

**主要类**：
- `myTello`：扩展的Tello类，提供视频帧获取和命令发送功能

### 3. TelloCV_ball_people.py
**功能**：
- 实现绿球跟踪功能
- 实现人脸跟踪功能
- 自动跟踪控制，根据目标位置自动调整无人机姿态

**主要类**：
- `TelloCV`：计算机视觉处理类，包含球跟踪和人脸跟踪功能

### 4. ball_tracker.py
**功能**：
- 实现绿球检测和跟踪算法
- 使用HSV颜色空间进行颜色检测
- 使用霍夫圆检测识别圆形目标
- 计算目标与无人机的距离

**主要类**：
- `ball_tracker`：球跟踪类，提供球检测和位置计算功能

### 5. face_tracker.py
**功能**：
- 基于深度学习的人脸检测
- 计算人脸与无人机的距离
- 实现人脸跟踪控制

**主要类**：
- `face_tracker`：人脸跟踪类，提供人脸检测和位置计算功能

### 6. mySpeechRecognition.py
**功能**：
- 实现语音录音功能
- 调用百度AI语音识别API进行语音识别
- 将识别的语音转换为无人机控制指令

**主要类**：
- `mySpeechRecognition`：语音识别类，提供录音和语音识别功能

### 7. mySpeechRecognition_UI.py
**功能**：
- 语音识别的图形界面
- 显示识别结果和已发送的命令
- 控制语音识别的开始和暂停

**主要类**：
- `mySpeechRecognition_UI`：语音识别界面类
- `speech_recognition_thread`：语音识别线程类

### 8. myGesture.py
**功能**：
- 调用百度AI手势识别API
- 识别10种手势（数字1-9、点赞、拳头等）
- 将识别的手势转换为无人机控制指令

**主要类**：
- `myGesture`：手势识别类，提供手势识别和指令转换功能

### 9. myGesture_UI.py
**功能**：
- 手势控制的图形界面
- 显示摄像头画面和识别结果
- 控制手势识别的开始和停止

**主要类**：
- `myGesture_UI`：手势识别界面类

### 10. AlphaPose_control.py
**功能**：
- 基于AlphaPose的人体姿态估计
- 支持10种人体姿态识别（双垂、双平举、双高举等）
- 实时关键点检测和可视化
- 姿态到控制指令的映射

**主要类**：
- `AlphaPose_control`：体态控制类，提供姿态估计和指令转换功能

### 11. myLoadingUI.py
**功能**：
- 程序启动时的加载界面
- 显示加载进度

**主要类**：
- `myLoading_UI`：加载界面类

## 环境要求

### 硬件要求
- Tello无人机
- 电脑（Windows系统）
- 摄像头（用于手势和体态识别）
- 麦克风（用于语音识别）

- Python 3.6+
- PyQt5

- PyAudio
- PyAV
- NumPy
- PyTorch
- 百度AI SDK

确保AlphaPose相关模型文件已下载到正确的目录。

## 运行步骤

### 1. 连接无人机
1. 打开Tello无人机电源
2. 在电脑上连接Tello的WiFi网络（Tello-XXXXXX）

### 2. 启动程序
```bash
python main.py
```

### 3. 使用说明

#### 3.1 基本操作
1. 点击界面上的"电源"按钮建立连接
2. 选择连接类型：
   - 无拓展板：标准Tello
   - 有拓展板：Tello EDU或带有RMTT拓展板的版本

#### 3.2 键盘控制
- `Tab`：起飞
- `L`：降落
- `P`：悬停
- `H`：在手上降落
- `W`：向前
- `S`：向后
- `A`：向左
- `D`：向右
- `U`：向上
- `I`：向下
- `J`：顺时针旋转
- `K`：逆时针旋转
- `Z`：前空翻
- `X`：后空翻
- `C`：左空翻
- `V`：右空翻
- `G`：抛飞模式

#### 3.3 语音控制
1. 点击"语音控制"按钮打开语音控制界面
2. 点击"开始"按钮开始语音识别
3. 说出指令，例如：
   - "起飞"
   - "降落"
   - "向前飞"
   - "向左飞"
   - "向上飞"
   - "顺时针旋转"
   - "悬停"
   等等

#### 3.4 手势控制
1. 点击菜单栏的"手势控制"打开手势控制界面
2. 点击"开始手势控制"按钮
3. 在摄像头前做出手势：
   - 拇指向上：起飞
   - 拇指向下：降落
   - 数字1：向上
   - 数字2：向下
   - 数字3：向左
   - 数字4：向右
   - 数字5：向前
   - 数字6：向后
   - 数字7：顺时针旋转90度
   - 数字8：逆时针旋转90度
   - 数字9：悬停
   - 拳头：在手上降落

#### 3.5 智能跟踪
1. 点击菜单栏的"绿球跟踪"开启绿球跟踪功能
2. 点击菜单栏的"人脸跟踪"开启人脸跟踪功能
3. 无人机会自动跟踪目标并调整位置

#### 3.6 体态控制
1. 点击菜单栏的"体态控制"开启体态控制功能
2. 在摄像头前做出特定姿态：
   - 双垂：双臂自然下垂
   - 双平举：双臂水平伸直
   - 双高举：双臂向上伸直
   - 双伸：双臂向前伸直
   - 左伸右平举：左臂向前，右臂水平
   - 左伸右垂：左臂向前，右臂下垂
   - 左平举右伸：左臂水平，右臂向前
   - 左垂右伸：左臂下垂，右臂向前
   - 左高举右垂：左臂向上，右臂下垂
   - 右高举左垂：右臂向上，左臂下垂

#### 3.7 拍照和录像
- 点击"拍照"按钮拍摄照片
- 点击"录像"按钮开始/停止录像

#### 3.8 调整参数
- 使用滑块调整飞行速度
- 使用滑块调整移动步长

### 4. 关闭程序
1. 点击"电源"按钮断开连接（无人机会自动降落）
2. 关闭程序窗口

## 注意事项

1. **安全第一**：
   - 首次使用请在开阔空间进行
   - 始终保持无人机在视线范围内
   - 准备好紧急制动按钮

2. **网络连接**：
   - 确保电脑已连接到Tello的WiFi
   - 保持网络稳定

3. **API配置**：
   - 确保百度AI API密钥正确配置
   - 注意API调用次数限制

4. **性能优化**：
   - 语音识别需要稳定的网络连接
   - 手势和体态识别需要良好的光线条件
   - 建议使用性能较好的电脑以保证流畅运行

5. **电量管理**：
   - 注意监控无人机电量
   - 低电量时及时降落

## 常见问题

### Q1: 无法连接到无人机
A: 
- 确认无人机电源已打开
- 确认电脑已连接到Tello的WiFi
- 尝试重启无人机和电脑

### Q2: 视频流显示卡顿
A:
- 检查网络连接质量
- 关闭其他占用网络的程序
- 降低视频质量设置

### Q3: 语音识别不准确
A:
- 确保麦克风工作正常
- 在安静环境下使用
- 清晰地说出指令

### Q4: 手势/体态识别不灵敏
A:
- 确保摄像头工作正常
- 保证良好的光线条件
- 保持适当距离（1-2米）
